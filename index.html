Sistem Informasi Geografis Penyebaran DBD

Website ini menampilkan peta interaktif untuk memantau penyebaran kasus demam berdarah di wilayah kerja Puskesmas Karya Maju. Dibuat sebagai proyek tugas akhir, sistem ini memungkinkan admin untuk input lokasi kasus dan masyarakat bisa melihat sebaran langsung di peta.

  • Tautan: GitHub
deteksi katarak App
Aplikasi Deteksi Katarak

Proyek ini merupakan aplikasi mobile yang dirancang untuk membantu mendeteksi potensi katarak dari citra mata pengguna. Ini adalah Capstone Project saya dalam program Bangkit Academy 2023, di mana saya berperan sebagai Project Manager sekaligus Backend Developer. Saya membangun RESTful API menggunakan Node.js dengan framework Hapi. Proyek ini juga memanfaatkan berbagai layanan cloud seperti Google Cloud Storage dan Cloud Run untuk memastikan kinerja yang optimal.

  • Teknologi: Node.js, Hapi, Machine Learning, Google Cloud Storage, Cloud Run, Firebase, Mobile App.
  • Tautan: GitHub
